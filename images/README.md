Папка для зображень галереї
============================

Щоб додати фото в галерею:

1. Скопіюйте ваші фото в цю папку (images/)
   - Підтримувані формати: JPG, JPEG, PNG, WEBP, GIF

2. Відредагуйте файл js/galerka.js
   - Додайте шлях до вашого фото в масив galleryImages
   - Приклад: 'images/your-photo.jpg'

3. Збережіть файл і перезавантажте сторінку

Фото з'являться в смужці прокрутки між Hero та Меню.
Можна гортати мишкою або сенсорним екраном (swipe).
